# AI Motion Lab

Первая версия портфолио для продажи AI-видео, motion design, 3D-визуалов, обложек и рекламных роликов.

Открыть сайт: `index.html`

Что внутри:
- `index.html` - страница портфолио.
- `styles.css` - стили без сборки и зависимостей.
- `assets/` - опубликованные видео, постеры и визуалы.

Сейчас на сайте 10 готовых spec-кейсов.

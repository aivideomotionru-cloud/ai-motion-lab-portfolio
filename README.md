# AI Motion Lab

Первая версия портфолио для продажи AI-видео, motion design, 3D-визуалов, обложек и рекламных роликов.

Открыть сайт: `index.html`

Что внутри:
- `index.html` - локальная страница портфолио.
- `styles.css` - стили без сборки и зависимостей.
- `assets/hero-ai-motion.png` - сгенерированный hero-визуал.
- `docs/portfolio-plan.md` - план demo/spec-кейсов.
- `docs/email-and-outreach.md` - рабочая почта и шаблоны писем.
